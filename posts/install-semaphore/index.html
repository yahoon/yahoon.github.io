Background#

I’m a long-time user of Ansible Automation Platform (AAP), and I deeply understand the importance of an automation platform. As the upstream of AAP, the open-source project AWX hasn’t been updated for a while, with the latest version still stuck in 2024.

This forced me to look for alternative tools.

Through an AI search, Semaphore UI stood out. It’s an AAP-like automation operations platform with a Web UI that supports not only Ansible but also Terraform and others.

Official website: https://semaphoreui.com/

Open source project: https://github.com/semaphoreui/semaphore

Semaphore#

Hosting Server#

I have my Desktop as the hosting server:

OS: Ubuntu 26.04 with podman installed

Install#

Per the GitHub homepage recommendation, install using containers.

To download other formats, refer to:

  • Installation wizard on the homepage: https://semaphoreui.com/install
  • Download various packages: https://github.com/semaphoreui/semaphore/releases

Open the Docker installation guide page https://semaphoreui.com/install/docker/2_18, fill in the customized options, and it will auto-generate the command.

The command it gave me:

docker run --name semaphore \
-p 3000:3000 \
-e SEMAPHORE_DB_DIALECT=sqlite \
-e SEMAPHORE_ADMIN=admin \
-e SEMAPHORE_ADMIN_PASSWORD=admin \
-e SEMAPHORE_ADMIN_NAME="Admin" \
-e SEMAPHORE_ADMIN_EMAIL=semaphore-admin@yahoon.io \
-v semaphore_data:/var/lib/semaphore \
-v semaphore_config:/etc/semaphore \
-v semaphore_tmp:/tmp/semaphore \
-d semaphoreui/semaphore:v2.18.0

Prepare Semaphore Image#

Since this is a homelab that can’t access Docker Hub, I first pulled the following image locally with a proxy enabled:

docker.io/semaphoreui/semaphore        latest      30f8744c37ca  2 days ago   944 MB

Just in case, push it to the local registry:

❯ podman tag docker.io/semaphoreui/semaphore:latest 192.168.1.4:5000/semaphoreui/semaphore:latest

❯ podman push 192.168.1.4:5000/semaphoreui/semaphore:latest

// The following commands are executed in a non-proxy Terminal environment

// The modified command is:

-p 3000:3000 \
-e SEMAPHORE_DB_DIALECT=sqlite \
-e SEMAPHORE_ADMIN=admin \
-e SEMAPHORE_ADMIN_PASSWORD=admin \
-e SEMAPHORE_ADMIN_NAME="Admin" \

-v semaphore_config:/etc/semaphore \

// I changed docker -> podman, and image version v2.18.0 -> latest

Access#

  1. From a client browser, visit http://192.168.1.4:3000
  2. Log in with the admin username and password
  3. Once logged in, follow the prompts to create a demo project
  4. Go to Task Templates to execute some tasks
